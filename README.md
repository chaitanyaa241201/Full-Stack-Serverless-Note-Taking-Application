# Project Overview
This project is a Full-Stack Serverless Note-Taking Application designed to facilitate easy note-taking in a digital environment.

# Architecture Diagrams
![](architecture_diagram.png)

# Tech Stack
- Frontend: React
- Backend: AWS Lambda, API Gateway
- Database: DynamoDB

# Features
- Create, read, update, and delete notes.
- User authentication.
- Responsive design.

# Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/chaitanyaa241201/Full-Stack-Serverless-Note-Taking-Application.git
   cd Full-Stack-Serverless-Note-Taking-Application
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
